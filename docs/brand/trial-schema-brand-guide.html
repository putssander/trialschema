Brand Guide

Trial Schema

A modern clinical AI brand system for structuring trial inclusion and exclusion criteria into agent-readable schemas.

Brand Essence

Trial Schema is a clinical-trial matching infrastructure tool that transforms inclusion and exclusion criteria into structured schemas for AI agents.

The brand should feel clinical, structured, intelligent, trustworthy, modern, precise, and developer-friendly.

Avoid generic healthcare clichés. The identity should feel like a high-quality health-tech SaaS platform, not a hospital or pharmaceutical brand.

Logo Concept

The logo communicates clinical trial logic, schema structure, and agent matching.

  • Three stacked input nodes represent trial criteria.
  • A checkmark represents inclusion.
  • An X represents exclusion.
  • A merge/filter shape represents schema transformation.
  • An output node represents the matched result.

Typography

Use Inter as the primary typeface.

  • Inter SemiBold for the logo wordmark
  • Inter Bold for headings
  • Inter Medium for navigation and buttons
  • Inter Regular for body text
font-family: "Inter", system-ui, sans-serif;
font-weight: 700;
letter-spacing: -0.03em;

Logo Usage

Primary
Website header, hero, docs, dashboards.
Icon Only
Sidebar, avatar, loading states.
Favicon
Use simplified icon only. No text.

Visual Style

Use flat vector geometry, rounded connectors, node-based diagrams, spacious white backgrounds, and subtle shadows.

Avoid caduceus symbols, stethoscopes, pills, hospital crosses, overdone DNA visuals, and cartoon robots.

Voice & Tone

Precise but not cold. Clinical but not sterile. Technical but not intimidating. Trustworthy but modern.
